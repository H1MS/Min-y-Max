# Min-y-Max

El proyecto consistia en crear un juego donde un jugador perseguia a otro, y el segundo jugador trata de huir. 
Pero antes de que comience la persecucion tenemos que crear un escenario.
Con el escenario creado, establecemos las reglas del juego, orden de turnos, movimientos, condiciones de victoria.
Como la idea era que el juego sea maquina contra maquina, creamos las funciones que permiten a la consola tomar decisiones.
